## Command palette

Press **⌘K** (Mac) or **Ctrl+K** (Windows/Linux) from anywhere in Alva to open the command palette.

The command palette lets you:

- **Search for anything** — companies, contacts, deals, pages
- **Navigate quickly** — jump to any page without clicking through menus
- **Start a conversation with Alva** — type a question and Alva takes over
- **Run quick actions** — create records, open settings, switch views

Start typing to filter results. Use the **arrow keys** to navigate and **Enter** to select.

## Available shortcuts

| Shortcut | Action |
|----------|--------|
| **⌘K** / **Ctrl+K** | Open command palette |
| **Esc** | Close the command palette or chat panel |

<Tip>
  The command palette is the fastest way to get around Alva. Instead of navigating through the sidebar, press ⌘K and type where you want to go — "deals", "settings", "Acme Corp" — and you're there in seconds.
</Tip>
